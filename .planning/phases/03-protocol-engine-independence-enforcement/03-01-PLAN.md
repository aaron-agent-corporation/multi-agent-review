## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| input document path -> filesystem | user-supplied path copied into a scoped workdir; must not escape runs/ |
| agent name -> filesystem path | agent name becomes a directory segment under work/; must be charset-gated |
| prompt/input content -> subprocess argv | content passed as execa array args, never a shell string |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | scope.ts agent-name -> work/<agent> path | mitigate | Validate agent against `/^[A-Za-z0-9_-]+$/`; throw on "/" or ".."; join only under runDir (Task 2 test asserts the throw) |
| T-03-02 | Information Disclosure | draft-phase workdir | mitigate | scopedWorkdir places ONLY input.md; peer draft physically absent; test asserts readdirSync excludes peer draft (the project's core confidentiality invariant) |
| T-03-03 | Tampering | execa cwd change to codex | mitigate | cwd is additive/optional; --skip-git-repo-check/--ephemeral/-s read-only retained; adapter-cwd test co-asserts pinned codex flags |
| T-03-04 | Elevation | prompt/input content as shell command | accept | execa array argv, no shell, stdin:"ignore" (EXISTING T-01-05/02-05) — unchanged by this plan |
| T-03-SC | Tampering | npm installs | mitigate | This plan adds NO new packages; install gate deferred to Plan 02 (xstate). No package-manager action here. |
