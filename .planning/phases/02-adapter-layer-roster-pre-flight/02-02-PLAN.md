## Tasks — CONTEXT.md decision citations

- **D-22:** Retry TRANSIENT failures only (timeouts, 429/RESOURCE_EXHAUSTED/usage-limit, unparseable-JSON flukes); NEVER auth or clean vendor errors — classifiers default unclassified errors to fatal. (Tasks 1, 2)
- **D-23:** Default 2 retries (3 attempts total), exponential backoff with jitter (~15s->~60s), honoring retry-after hints. (Task 2)
- **D-24:** ONE vendor-agnostic `withRetry` wrapper around `AgentAdapter.invoke` — not duplicated per adapter. (Task 2)
- **D-25:** Every attempt (including failed ones) gets its own `invocations.ndjson` record with an attempt number. (Tasks 1, 2)
- **D-35:** No new dependency — plain TS retry via `node:timers/promises`; p-retry rejected. (Task 2)
