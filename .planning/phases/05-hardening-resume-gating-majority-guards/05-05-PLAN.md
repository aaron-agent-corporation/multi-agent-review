<objective>
The gating vertical slice (PROT-05 + the gated-arbitration half of RSLV-03): a run-start mode prompt
(autonomous vs gated, D-53) guarded by `isTTY` with a non-TTY default-autonomous bypass (Pitfall 5),
explicit mode/pause flags, a blocking phase-boundary gate (approve / abort / feedback, D-50/D-51) with

resume` (05-04, D-55), feedback injected into the NEXT phase's prompts only, and gated arbitration of
an escalated disagreement recorded as `resolver: human` (D-52). All interaction rides an injectable
`ask()` seam so tests never depend on real TTY.

Purpose: PROT-05 requires a per-run autonomous/gated choice with pauses at phase boundaries; RSLV-03
requires escalations to pause for human arbitration in gated mode and log-only in autonomous mode.
Output: the mode prompt + flags + ask seam (cli.ts), the gate hooks + paused state + feedback +
arbitration (engine.ts), a `mode` config default (config.ts), and gating tests via the seam.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| human feedback note → next-phase prompt | a human-authored note is injected into agent prompts — must not break the thin-prompt contract or inject the format vocabulary; treated as steering only |
| stdin / TTY → run control | a scripted/non-TTY run must never block on a prompt |
| gated arbitration ruling → decision record | a human ruling becomes a resolved decision feeding the re-litigation guard |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-14 | Tampering | human feedback note injection into prompts (steering escalating into format/contract override) | mitigate | feedback is a short note appended to the NEXT phase prompt only, never edits an artifact; the seeded format contract remains the sole format authority (thin-prompt convention preserved); note stored with attribution |
| T-05-15 | Denial of Service | a blocking gate prompt hangs a scripted/CI run | mitigate | isTTY guard + non-TTY default-autonomous; pause-and-exit alternative; ask() seam (Pitfall 5) |
| T-05-16 | Repudiation | a human arbitration ruling recorded without rationale/attribution | mitigate | arbitration records `resolver:"human"` with the human's rationale into the decision record/ledger; gate-feedback notes carry attribution+timestamp |
</threat_model>
